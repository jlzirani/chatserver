chatserver
==========

A simple chat server monothread and multiuser in c for linux.

How to compile
==============

$> mkdir bin obj
$> make

How to run
==========

(Rem: the chatserver will listen on 0.0.0.0:4000)

  $> bin/chatserver <br/>
or with a normal output :<br/>
  $> bin/chatserver -d <br/>
or withe a colored output :<br/>
  $> bin/chatserver -d color
